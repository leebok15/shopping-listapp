# 🛒 쇼핑 리스트

간단한 바닐라 JavaScript 쇼핑 리스트 앱입니다. 별도의 빌드 과정이나 프레임워크 없이 HTML, CSS, JavaScript만으로 작성되었습니다.

## 기능

- 아이템 추가
- 아이템 클릭 또는 체크박스로 완료 표시
- 아이템 삭제
- 체크된 항목 일괄 삭제
- 총 개수 및 완료 개수 요약 표시
- `localStorage`를 이용한 데이터 저장 (새로고침해도 유지됨)
- 라이트/다크 모드 자동 대응

## 실행 방법

별도의 설치 과정 없이 `index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

## 파일 구성

| 파일 | 설명 |
| --- | --- |
| `index.html` | 앱의 마크업 구조 |
| `style.css` | 스타일 및 다크 모드 대응 |
| `script.js` | 아이템 추가/토글/삭제 등 앱 로직 |
